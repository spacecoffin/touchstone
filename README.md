# Touchstone
_a test or criterion for determining the quality or genuineness of a thing_

Touchstone provides scalable machine learning metrics.

### Regressions Metrics
- R Squared
- Mean Squared Error

### Classification Metrics
- Matthews Correlation Coefficient
- F Score
- Confusion Matrix
